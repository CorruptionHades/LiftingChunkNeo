# LiftingChunksNeo

This is a mod recreating BastiGHG's (latest) [challenge](https://www.youtube.com/watch?v=LPIyGLzedxE).

### Version
- Minecraft 1.21.1

### Dependencies:
- [Sable](https://github.com/ryanhcode/sable) [Download](https://modrinth.com/mod/sable/)
- [Neoforge](https://neoforged.net/)

### Usage:
`/liftingchunks start` - Start the challenge.

`/liftingchunks stop` - Stop the challenge.

`/liftingchunks pause` - Pause the challenge.
